# Rust 向量 Vec\<T\> 基础

`Vec<T>` 是 Rust 中最常用的动态数组，它在堆上分配一块连续内存，存放相同类型的元素。`Vec<T>` 拥有数据的所有权，当离开作用域时会自动释放内存。

> 进阶操作（联结/拆分/交换/排序/搜索等）请参见：[向量进阶](向量进阶.md)。

## 一、速览

| 特性 | 说明 |
| :--- | :--- |
| 存储位置 | 堆 |
| 长度 | 动态可变，运行时增长或缩短 |
| 元素类型 | 必须相同，编译时确定 |
| 是否拥有数据 | 是，拥有所有权 |
| 默认值 | 无（通过 `Vec::new()` 创建空向量） |

## 二、基本操作

```rust
    // 创建一个空向量
    let mut v1: Vec<i32> = Vec::new();

    // 使用 vec! 宏，带初始值
    let v2 = vec![1, 2, 3];

    // 创建指定长度和初始值的向量
    let v3 = vec![0; 5];  // [0, 0, 0, 0, 0]

    // 从迭代器收集
    let v4: Vec<_> = (0..10).collect();

    let mut v = vec![1, 2, 3];

    // 添加元素
    v.push(4);  // 向末尾添加元素
    v.pop();    // 移除并返回最后一个元素，返回 Option<T>

    // 访问元素
    let first = &v[0];            // 索引访问，越界会 panic
    let second = v.get(1);        // 安全访问，返回 Option<&T>

    // 修改元素
    v[0] = 10;

    // 遍历
    for i in &v {
        println!("{}", i);
    }

    // 带索引的遍历
    for (i, val) in v.iter().enumerate() {
        println!("v[{}] = {}", i, val);
    }
```

## 三、内存布局

`Vec<T>` 在栈上保存三个信息：
    - 指向堆上连续数组的指针
    - 当前元素数量（长度）
    - 当前分配的容量（cap）

当 `push` 导致长度超过容量时，`Vec` 会自动在堆上分配一块更大的内存（通常是原容量的两倍），将现有元素拷贝过去，再释放旧内存。这种自动扩容机制使得 `Vec` 既高效又灵活。

## 四、容量相关方法

```rust
    let mut v: Vec<i32> = Vec::new();

    v.capacity();  // 当前分配的容量
    v.len();       // 当前元素个数

    v.reserve(10); // 预留额外空间，避免频繁分配
    v.shrink_to_fit(); // 收缩容量到实际长度

    v.is_empty();  // 判断是否为空
```

## 五、删除元素

```rust
    let mut v = vec![1, 2, 3, 4, 5];

    // 移除最后一个元素
    let last = v.pop();  // Some(5)

    // 移除指定索引的元素，后面的元素会向前移动
    let elem = v.remove(1); // 移除索引1处的2，返回2

    // 清空所有元素
    v.clear();
```

## 六、与切片 `&[T]` 的关系

`Vec<T>` 可以自动解引用为切片 `&[T]`，这使得任何接受切片引用的函数都可以直接传入 `&Vec<T>`。

```rust
    fn sum(slice: &[i32]) -> i32 {
        slice.iter().sum()
    }

    let v = vec![1, 2, 3];
    let total = sum(&v);  // &Vec<i32> 自动转换为 &[i32]

    // 也可以从 `Vec<T>` 创建切片：

    let v = vec![10, 20, 30, 40];
    let slice: &[i32] = &v[1..3];  // [20, 30]
```

切片只是视图，不拥有数据。更多细节请参考：[切片](/指针类型/切片.md)

## 七、常用迭代器适配器

```rust
    // `Vec<T>` 提供了丰富的迭代器方法用于转换和处理数据：

    let v = vec![1, 2, 3, 4, 5];

    // 映射
    let doubled: Vec<i32> = v.iter().map(|x| x * 2).collect();

    // 过滤
    let even: Vec<i32> = v.iter().filter(|&&x| x % 2 == 0).copied().collect();

    // 求和
    let sum: i32 = v.iter().sum();

    // 折叠
    let product = v.iter().fold(1, |acc, x| acc * x);
```

## 八、访问元素

除了索引访问外，切片（数组、向量也一样）还提供了安全便捷的元素访问方法。

```rust
    let v = vec![1, 2, 3, 4, 5];

    // 返回对第一个/最后一个元素的引用，返回 Option<&T>
    let first = v.first();     // Some(&1)
    let last = v.last();       // Some(&5)

    // 安全索引访问，返回 Option<&T>，越界时返回 None
    assert_eq!(v.get(2), Some(&3));
    assert_eq!(v.get(10), None);

    // 可变版本
    let mut v = vec![1, 2, 3];
    if let Some(last) = v.last_mut() {
        *last = 100;
    }
    assert_eq!(v, vec![1, 2, 100]);

    // 可变安全访问
    if let Some(elem) = v.get_mut(0) {
        *elem = 10;
    }
```

`to_vec()` 克隆整个切片，返回一个新向量：

```rust
    let arr = [1, 2, 3, 4, 5];
    assert_eq!(arr[0..3].to_vec(), vec![1, 2, 3]);
```
