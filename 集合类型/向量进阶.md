# Rust 向量 Vec\<T\> 进阶

> 前置知识：Vec 的基本操作、内存布局、容量管理请先阅读 [向量基础](向量基础.md)。

---

## 一、扩大与收缩

数组和切片一旦创建就无法调整大小，以下方法仅适用于 `Vec<T>`。

```rust
    let mut v = vec![1, 2, 3];

    // 在末尾添加/移除
    v.push(4);           // [1, 2, 3, 4]
    let last = v.pop();  // Some(4)

    // 在任意位置插入/移除（O(n) 操作，需要平移元素）
    v.insert(0, 0);      // [0, 1, 2, 3]
    let elem = v.remove(1); // 移除索引1处的元素，返回1
```

> **注意**：`insert` 和 `remove` 需要移动的元素越多速度就越慢。如果经常在首端操作，请考虑使用 [VecDeque](./VecDeque.md)。

**调整长度：**

```rust
    let mut v = vec![1, 2, 3];

    // resize：调整长度，新空间用 value 的副本填充（要求 T: Clone）
    v.resize(5, 0);   // [1, 2, 3, 0, 0]

    // resize_with：用闭包构造新元素（不要求 Clone）
    v.resize_with(5, || 42);  // [1, 2, 3, 0, 0]（假设前面已 resize）

    // truncate：截断到指定长度，丢弃多余元素
    v.truncate(2);     // 保留前2个

    // clear：清空所有元素，等效于 truncate(0)
    v.clear();
```

**批量添加/移除：**

```rust
    let mut v = vec![1, 2, 3];

    // extend：在末尾批量添加
    v.extend([4, 5, 6].iter().copied());  // [1, 2, 3, 4, 5, 6]

    // append：将另一个向量中的所有元素移动到末尾，之后 vec2 被清空
    let mut v2 = vec![7, 8];
    v.append(&mut v2);  // v = [1, 2, 3, 4, 5, 6, 7, 8], v2 = []

    // split_off：在指定索引处拆分，返回后半部分的新向量
    let mut v = vec![1, 2, 3, 4];
    let tail = v.split_off(2);  // v = [1, 2], tail = [3, 4]

    // drain：移除指定范围，返回消耗型迭代器
    let mut v = vec![1, 2, 3, 4, 5];
    let drained: Vec<_> = v.drain(1..4).collect();  // [2, 3, 4], v = [1, 5]
```

**选择性移除：**

```rust
    // retain：保留满足条件的元素
    let mut v = vec![1, 2, 3, 4, 5];
    v.retain(|&x| x % 2 == 0);  // [2, 4]

    // dedup：移除相邻重复元素（类似 Unix uniq）
    let mut v = b"Misssssssissippi".to_vec();
    v.dedup();
    assert_eq!(&v, b"Misisipi");  // 注意：输出中仍有两个 's'，因为 dedup 只移除相邻重复

    // dedup_by：使用自定义比较函数
    v.dedup_by(|a, b| a == b);

    // dedup_by_key：按键去重
    errors.dedup_by_key(|err| err.to_string());
```

> `retain` 的性能优于 `v = v.into_iter().filter(test).collect()`，因为它原地操作，不需要重新分配。

## 二、联结

以下方法适用于「数组的数组」——元素本身也是数组、切片或向量的情况。

```rust
    // concat：串联所有切片，返回新向量
    assert_eq!([[1, 2], [3, 4], [5, 6]].concat(),
               vec![1, 2, 3, 4, 5, 6]);

    // join：在切片之间插入分隔符
    assert_eq!([[1, 2], [3, 4], [5, 6]].join(&0),
               vec![1, 2, 0, 3, 4, 0, 5, 6]);
```

## 三、拆分

Rust 提供了丰富的切片拆分方法，可以安全地同时获取多个不重叠区域的引用。

**基本拆分：**

```rust
    let v = vec![0, 1, 2, 3, 4, 5];

    // split_at：在索引处分成两半
    let (left, right) = v.split_at(3);
    // left = &[0, 1, 2], right = &[3, 4, 5]

    // split_first / split_last：分离首/尾元素
    if let Some((first, rest)) = v.split_first() {
        // first = &0, rest = &[1, 2, 3, 4, 5]
    }
    if let Some((last, rest)) = v.split_last() {
        // last = &5, rest = &[0, 1, 2, 3, 4]
    }
```

**按分隔符拆分：**

```rust
    let v = [1, 2, 0, 3, 4, 0, 5];

    // split：按分隔符拆分，分隔符不包含在输出中
    let parts: Vec<_> = v.split(|&x| x == 0).collect();
    // [[1, 2], [3, 4], [5]]

    // split_inclusive：分隔符包含在前一个子切片的末尾
    let parts: Vec<_> = v.split_inclusive(|&x| x == 0).collect();
    // [[1, 2, 0], [3, 4, 0], [5]]

    // rsplit：从右向左拆分
    let parts: Vec<_> = v.rsplit(|&x| x == 0).collect();
    // [[5], [3, 4], [1, 2]]

    // splitn：最多拆分为 n 片
    let parts: Vec<_> = v.splitn(2, |&x| x == 0).collect();
    // [[1, 2], [3, 4, 0, 5]]
```

**按块拆分：**

```rust
    let v = [1, 2, 3, 4, 5];

    // chunks：分成大小为 n 的非重叠块
    let chunks: Vec<_> = v.chunks(2).collect();
    // [[1, 2], [3, 4], [5]]

    // chunks_exact：精确分块，余数通过 remainder() 获取
    let mut iter = v.chunks_exact(2);
    // iter.next() => Some(&[1, 2])
    // iter.next() => Some(&[3, 4])
    // iter.remainder() => &[5]

    // rchunks / rchunks_exact：从末尾开始分块

    // windows：滑动窗口，生成 n 个连续元素的子切片
    let windows: Vec<_> = v.windows(2).collect();
    // [[1, 2], [2, 3], [3, 4], [4, 5]]
```

> 滑动窗口可用于分析数据序列的变化：`temperatures.windows(2).map(|w| w[1] - w[0])`

以上所有方法都有对应的 `_mut` 版本（`windows` 除外，因为子切片之间有重叠）。

## 四、交换

```rust
    let mut v = vec![1, 2, 3, 4];

    // swap：交换两个索引处的元素
    v.swap(0, 3);  // [4, 2, 3, 1]

    // swap_with_slice：交换两个等长切片的全部内容
    let mut a = [1, 2, 3];
    let mut b = [4, 5, 6];
    a.swap_with_slice(&mut b);
    // a = [4, 5, 6], b = [1, 2, 3]

    // swap_remove：移除指定索引的元素，将最后一个元素移到空缺处
    // 与 remove 不同，不保持剩余元素的顺序，但 O(1)
    let mut v = vec!["a", "b", "c", "d"];
    let removed = v.swap_remove(1);  // 移除"b"，"d"移到它的位置
    // v = ["a", "d", "c"]
```

## 五、填充

```rust
    let mut v = vec![0; 5];

    // fill：用 value 的克隆体填充（要求 T: Clone）
    v.fill(42);  // [42, 42, 42, 42, 42]

    // fill_with：用闭包生成的值填充（不要求 Clone）
    // 对于 Option<T> 等未实现 Clone 的类型很有用
    v.fill_with(|| 7);  // [7, 7, 7, 7, 7]
```

## 六、排序与搜索

**排序：**

```rust
    let mut v = vec![3, 1, 4, 1, 5, 9];

    // sort：升序排序（要求 T: Ord）
    v.sort();  // [1, 1, 3, 4, 5, 9]

    // sort_by：按自定义比较函数排序
    v.sort_by(|a, b| b.cmp(a));  // 降序

    // sort_by_key：按键排序
    students.sort_by_key(|s| s.grade_point_average());
```

> **注意**：`sort_by_key` 的键函数不能返回借用于元素的引用（`|s| &s.name` 无法编译），此时可使用 `sort_by` 作为替代。

排序后可用 `reverse()` 逆转：

```rust
    v.reverse();  // 就地逆转
```

**二分搜索（要求切片已排序）：**

```rust
    let v = vec![1, 3, 5, 7, 9];

    // binary_search：返回 Result<usize, usize>
    assert_eq!(v.binary_search(&5), Ok(2));     // 找到，返回索引
    assert_eq!(v.binary_search(&4), Err(2));     // 未找到，返回插入点

    // binary_search_by / binary_search_by_key 提供自定义比较
```

**线性搜索：**

```rust
    // contains：检查是否包含某个值
    assert!(v.contains(&3));

    // position：找到值的位置（类似 JavaScript 的 indexOf）
    let pos = v.iter().position(|&x| x == 5);  // Option<usize>
```

> `f32` 和 `f64` 因 NaN 无法实现 `Ord`，不能直接用于排序和二分搜索。可使用 `ord_subset` crate。

## 七、比较切片

切片之间支持按字典序比较（要求元素实现 `PartialEq` / `PartialOrd`）：

```rust
    // 相等比较
    assert_eq!([1, 2, 3] == [1, 2, 3], true);

    // 字典序比较
    assert!([1, 2, 3] < [1, 2, 4]);
```

便捷的前缀/后缀检查：

```rust
    assert_eq!([1, 2, 3, 4].starts_with(&[1, 2]), true);
    assert_eq!([1, 2, 3, 4].ends_with(&[3, 4]), true);
```

## 八、随机元素

随机数功能由 `rand` crate 提供（非标准库）：

```rust
    use rand::seq::SliceRandom;
    use rand::thread_rng;

    let v = vec![1, 2, 3, 4, 5];

    // choose：随机选取一个元素的引用
    let random = v.choose(&mut thread_rng());  // Option<&T>

    // shuffle：就地随机洗牌
    let mut v = vec![1, 2, 3, 4, 5];
    v.shuffle(&mut thread_rng());
```

## 九、Rust 中不存在失效型错误

许多主流语言（Python、Java、C++ 等）中，在迭代集合的同时修改它会导致**失效型错误**（迭代器失效）。

```rust
    // 这段代码无法通过编译！
    // let mut my_vec = vec![1, 3, 5, 7, 9];
    // for (index, &val) in my_vec.iter().enumerate() {
    //     if val > 4 {
    //         my_vec.remove(index);  // 错误：不能把 my_vec 借用为可变的
    //     }
    // }
```

Rust 在编译时就能捕获此类错误：调用 `my_vec.iter()` 时借用了共享（不可变）引用，迭代期间不能修改向量。

正确的做法：

```rust
    let mut my_vec = vec![1, 3, 5, 7, 9];

    // 方案1：使用 retain
    my_vec.retain(|&val| val <= 4);

    // 方案2：用 filter 创建新向量
    let filtered: Vec<_> = my_vec.iter().filter(|&&val| val <= 4).copied().collect();
```
