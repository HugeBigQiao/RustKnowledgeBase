# Rust 循环表达式

Rust 提供了三种循环结构：`loop`、`while` 和 `for`。其中只有 `loop` 是真正的**表达式**，可以通过 `break` 返回一个值。`while` 和 `for` 不是表达式，固定返回单元类型 `()`。

## 一、`loop` 表达式

`loop` 会无限循环执行代码块中的内容，直到遇到 `break` 才退出。

```rust
let mut counter = 0;

loop {
    counter += 1;
    println!("第 {} 次循环", counter);
    if counter >= 5 {
        break;
    }
}
```

`break` 可以携带一个值，这个值会成为整个 `loop` 表达式的返回值。

```rust
let mut counter = 0;
let result = loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;
    }
};
// result == 20
```

`break` 后面的值不需要加分号，直接写在 `break` 之后即可。

## 二、`while` 循环

`while` 在每次循环开始前检查条件，条件为 `true` 时执行循环体。它**不是表达式**，不能通过 `break` 返回值。

```rust
let mut n = 3;
while n > 0 {
    println!("{}", n);
    n -= 1;
}
// while 整体返回单元类型 ()
```

如果需要在 `while` 循环中获取计算结果，通常使用外部变量保存。

```rust
let mut i = 0;
let mut found = None;
while i < 10 {
    if i == 5 {
        found = Some(i);
        break;
    }
    i += 1;
}
// found == Some(5)
```

## 三、`for` 循环

`for` 用于遍历实现了 `IntoIterator` trait 的类型，如数组、向量、切片、范围等。它同样**不是表达式**。

```rust
let arr = [10, 20, 30, 40, 50];
for element in &arr {
    println!("值为: {}", element);
}
```

使用范围表达式可以替代传统的计数循环。

```rust
// 0 到 4
for i in 0..5 {
    println!("{}", i);
}

// 反转范围
for i in (0..5).rev() {
    println!("{}", i);
}
```

遍历时如果需要索引，可以使用 `enumerate()` 方法。

```rust
let arr = ['a', 'b', 'c'];
for (index, value) in arr.iter().enumerate() {
    println!("arr[{}] = {}", index, value);
}
```

## 四、循环标签

当存在嵌套循环时，可以为外层循环打上标签，使用 `break 'label` 跳出指定层级的循环。

```rust
'outer: loop {
    println!("外层循环开始");
    loop {
        println!("内层循环");
        break 'outer; // 直接跳出外层循环
    }
}
// 程序继续从这里执行
```

`continue` 同样可以配合标签使用，跳到指定循环的下一次迭代。

```rust
'outer: for i in 0..3 {
    for j in 0..3 {
        if i == 1 && j == 1 {
            continue 'outer; // 跳到外层循环的下一次迭代
        }
        println!("i: {}, j: {}", i, j);
    }
}
```

## 五、`loop` 与 `match` 结合

`loop` 常用于需要重试或持续尝试直到成功的场景，与 `match` 结合使用可以优雅地处理不同情况。

```rust
let mut attempts = 0;
let result = loop {
    attempts += 1;
    if attempts > 3 {
        break "重试次数已用完";
    }
    // 模拟某种可能成功的操作
    if attempts == 2 {
        break "操作成功";
    }
};
// result == "操作成功"
```

## 六、三种循环的适用场景

- `loop`：无限循环、需要从循环中返回值、重试逻辑
- `while`：条件驱动循环，循环次数不确定，条件在每次迭代前检查
- `for`：遍历集合或范围，循环次数确定或由迭代器决定

```rust
// loop：重试直到成功
let mut connected = false;
loop {
    if connected {
        break;
    }
    // 尝试连接...
    connected = true;
}

// while：读取直到满足条件
let mut input = String::new();
while input.trim() != "quit" {
    // 读取输入...
    input = String::from("quit");
}

// for：遍历固定范围
for i in 0..10 {
    println!("{}", i);
}
```

## 速记要点

- `loop` 是表达式，可通过 `break value` 返回值
- `while` 和 `for` 不是表达式，固定返回 `()`
- `for` 用于遍历任何实现了 `IntoIterator` 的类型
- 循环标签用于在嵌套循环中精确控制 `break` 和 `continue`
- 计数循环用 `0..n` 范围配合 `for`，而不是传统三段式 `for`
