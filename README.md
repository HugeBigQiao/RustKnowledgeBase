# 🦀 Rust 知识库

本仓库是系统化的 Rust 知识库，以 Markdown 卡片形式组织，基于《Rust 程序设计（第 2 版）》一书内容整理。涵盖从基本类型到并发异步、从宏到不安全代码的完整 Rust 知识体系。所有文档均带有 YAML Frontmatter，便于构建 RAG 应用或生成文档站。

## 目录结构

```
基础类型/          整数 · 浮点 · 布尔 · 字符 · 元组 · 数组 · 类型别名
指针类型/          Box · Rc与Arc · 裸指针 · 切片
集合类型/          向量基础 · 向量进阶 · 字符串 · VecDeque · BinaryHeap · HashMap与BTreeMap · HashSet与BTreeSet
核心概念/          所有权 · 引用与借用

特型与泛型/        特型基础 · 泛型 · 关联类型 · 运算符重载 · 内存管理特型 · 类型转换特型
函数与闭包/        闭包 · 迭代器
控制流/            表达式总览 · if表达式 · match表达式 · 循环表达式 · return

```
