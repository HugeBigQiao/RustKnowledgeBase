# 迭代器

**迭代器**是一个值，可以生成一系列值。`for` 循环为迭代器提供了自然语法，迭代器本身也提供了丰富的适配器方法（`map`、`filter`、`fold`、`collect` 等）。迭代器是 Rust 中的零成本抽象——灵活、富有表现力且高效。

---

## 一、Iterator 特型与 IntoIterator 特型

### 1.1 Iterator

```rust
trait Iterator {
    type Item;
    fn next(&mut self) -> Option<Self::Item>;
    // ... 众多默认方法
}
```

- `Item`：迭代器生成的值的类型
- `next`：返回 `Some(v)` 或 `None`（序列结束）

### 1.2 IntoIterator

```rust
trait IntoIterator where Self::IntoIter: Iterator<Item=Self::Item> {
    type Item;
    type IntoIter: Iterator;
    fn into_iter(self) -> Self::IntoIter;
}
```

实现了 `IntoIterator` 的类型称为**可迭代者**。

### 1.3 for 循环的真相

```rust
for element in &v {
    println!("{}", element);
}

// 等价于：
let mut iterator = (&v).into_iter();
while let Some(element) = iterator.next() {
    println!("{}", element);
}
```

### 1.4 三种迭代方式

```rust
for element in &collection { ... }       // 共享引用迭代
for element in &mut collection { ... }   // 可变引用迭代
for element in collection { ... }        // 按值迭代（消耗集合）
```

每种最终调用不同的 `IntoIterator` 实现。所有迭代器也自动实现 `IntoIterator`。

## 二、创建迭代器

### 2.1 iter 与 iter_mut

大多数集合提供 `iter()` 和 `iter_mut()`，返回生成共享/可变引用的迭代器：

```rust
let v = vec![4, 20, 12, 8, 6];
let mut iterator = v.iter();
assert_eq!(iterator.next(), Some(&4));
```

### 2.2 into_iter

```rust
let mut favorites = BTreeSet::new();
favorites.insert("Lucy in the Sky With Diamonds".to_string());
let mut it = favorites.into_iter();
assert_eq!(it.next(), Some("Liebesträume No. 3".to_string()));
```

### 2.3 from_fn 与 successors

从闭包生成序列：

```rust
use std::iter::from_fn;

// 无穷随机数流 + take 限制
let lengths: Vec<f64> = from_fn(|| Some((random::<f64>() - random::<f64>()).abs()))
    .take(1000)
    .collect();

// 斐波那契数列
fn fibonacci() -> impl Iterator<Item=usize> {
    let mut state = (0, 1);
    std::iter::from_fn(move || {
        state = (state.1, state.0 + state.1);
        Some(state.0)
    })
}
```

`successors`：每个条目依赖前一个条目：

```rust
use std::iter::successors;
successors(Some(zero), |&z| Some(z * z + c))  // 曼德博集序列
```

### 2.4 drain

`drain` 借入可变引用，将元素所有权传给消费者，迭代器丢弃时清空集合：

```rust
let mut outer = "Earth".to_string();
let inner = String::from_iter(outer.drain(1..4));
assert_eq!(outer, "Eh");
assert_eq!(inner, "art");
```

### 2.5 其他迭代器源（速查）

| 来源 | 表达式 | 说明 |
|------|--------|------|
| 范围 | `1..10`, `1..=10` | 整数范围 |
| Option | `Some(10).iter()` | 0或1个元素 |
| Result | `Ok("blah").iter()` | 生成 Ok 值 |
| 切片 | `v.windows(16)`, `v.chunks(16)` | 窗口/分块 |
| 字符串 | `s.bytes()`, `s.chars()`, `s.lines()` | 按字节/字符/行 |
| 自由函数 | `empty()`, `once(5)`, `repeat("#9")` | 空/单次/重复 |

## 三、迭代器适配器

适配器消耗迭代器并构建新迭代器。**关键特性：惰性求值**——调用适配器不消耗条目，只有最终调用 `next` 时才工作。

### 3.1 map 与 filter

```rust
let text = "  ponies  \n   giraffes\niguanas  \nsquid".to_string();
let v: Vec<&str> = text.lines()
    .map(str::trim)
    .filter(|s| *s != "iguanas")
    .collect();
```

- `map(f)`：对每个条目应用 `f`，返回 `impl Iterator<Item=B>`
- `filter(p)`：保留 `p` 返回 `true` 的条目

### 3.2 filter_map 与 flat_map

`filter_map`：闭包返回 `Option<B>`，`None` 被丢弃，`Some(b)` 传出：

```rust
text.split_whitespace()
    .filter_map(|w| f64::from_str(w).ok())
```

`flat_map`：闭包返回可迭代者，将所有子序列串联：

```rust
countries.iter().flat_map(|country| &major_cities[country])
```

### 3.3 flatten

串联可迭代者的可迭代者。也可用于过滤 `Option`：

```rust
vec![None, Some("day"), None, Some("one")]
    .into_iter().flatten().collect::<Vec<_>>()  // ["day", "one"]
```

### 3.4 take / take_while / skip / skip_while

```rust
iter.take(n)              // 最多取n个
iter.take_while(|l| !l.is_empty())  // 遇到false时停止
iter.skip(1)              // 跳过前n个（如跳过程序名）
iter.skip_while(|l| !l.is_empty())  // 跳过直到闭包返回false
```

### 3.5 peekable

允许"窥视"下一个条目而不消耗：

```rust
let mut chars = "226153980,1766319049".chars().peekable();
fn parse_number<I>(tokens: &mut Peekable<I>) -> u32
    where I: Iterator<Item=char>
{
    match tokens.peek() {
        Some(r) if r.is_digit(10) => { /* 消耗数字 */ }
        _ => return n  // 非数字，留给调用者
    }
}
```

### 3.6 fuse

确保首次返回 `None` 后永远返回 `None`（处理行为异常的迭代器）。

### 3.7 可逆迭代器与 rev

实现了 `DoubleEndedIterator` 的迭代器支持 `next_back` 和 `rev`：

```rust
let meals = ["breakfast", "lunch", "dinner"];
let mut iter = meals.iter().rev();
assert_eq!(iter.next(), Some(&"dinner"));
```

### 3.8 chain、enumerate、zip

```rust
(1..4).chain([20, 30, 40])      // 串联：1,2,3,20,30,40
iter.enumerate()                  // 附加索引：生成 (0, A), (1, B), ...
(0..).zip("ABCD".chars())        // 拉合两个迭代器
```

### 3.9 by_ref

借入迭代器的可变引用，用完适配器后可继续使用原迭代器：

```rust
let mut lines = message.lines();
for header in lines.by_ref().take_while(|l| !l.is_empty()) { ... }
// lines 仍可使用
for body in lines { ... }
```

### 3.10 cloned / copied / cycle

```rust
a.iter().cloned()    // 克隆引用目标：&T → T (需Clone)
a.iter().copied()    // 解引用复制：&T → T (需Copy)
dirs.iter().cycle()  // 无限循环重复
```

## 四、消耗迭代器

### 4.1 简单累加

```rust
iter.count()     // 条目计数
iter.sum()       // 求和（整数/浮点）
iter.product()   // 求积
iter.min()       // 最小值（需Ord）
iter.max()       // 最大值（需Ord）
```

浮点类型未实现 `Ord`，用 `max_by`/`min_by` 代替：

```rust
numbers.iter().copied().max_by(|a, b| a.partial_cmp(b).unwrap())
```

### 4.2 max_by_key / min_by_key

```rust
populations.iter().max_by_key(|&(_name, pop)| pop)  // 人口最多的城市
```

### 4.3 条目序列比较

```rust
packed.split_whitespace().eq(spaced.split_whitespace())  // 逐词相等
spaced.split_whitespace().gt(obscure.split_whitespace())  // 逐词大于
```

### 4.4 any / all

```rust
id.chars().any(char::is_uppercase)   // 存在大写？
id.chars().all(char::is_uppercase)   // 全部大写？
```

只消耗确定答案所需的最少条目。

### 4.5 position / rposition

```rust
text.chars().position(|c| c == 'e')    // 首次出现位置
bytes.iter().rposition(|&c| c == b'e') // 从右侧首次出现位置（需双端+确切大小）
```

### 4.6 fold / rfold

通用的累加操作：

```rust
a.iter().fold(0, |n, i| n + i)            // 求和
a.iter().fold(String::new(), |s, w| s + w + " ")  // 拼接字符串
a.iter().rfold(String::new(), |s, w| s + w + " ") // 反向拼接
```

### 4.7 try_fold / try_rfold

支持提前退出的折叠。闭包返回 `Result`/`Option`/`ControlFlow`：

```rust
stdin.lock().lines()
    .try_fold(0, |sum, line| -> Result<u64, Box<dyn Error>> {
        Ok(sum + u64::from_str(&line?.trim())?)
    })?
```

### 4.8 nth / nth_back / last

```rust
squares.nth(4)        // 跳过4个，返回第5个
squares.nth_back(0)   // 最后一个（双端迭代器）
squares.last()        // 最后一个（消耗所有条目）
```

### 4.9 find / find_map

```rust
populations.iter().find(|&(_name, &pop)| pop > 500_000)  // 找第一个超50万的城市
iter.find_map(|item| { ...; Some(value) })  // 查找并映射
```

### 4.10 collect 与 FromIterator

`collect` 是通用方法，可以构建任意实现了 `FromIterator` 的集合：

```rust
let args: Vec<String> = std::env::args().collect();
let args: HashSet<String> = std::env::args().collect();
let args: HashMap<String, usize> = std::env::args().zip(0..).collect();
```

`Iterator::size_hint()` 提供条目数上下限，`Vec` 据此预分配正确大小的缓冲区。

### 4.11 Extend

扩展现有集合：

```rust
let mut v: Vec<i32> = (0..5).map(|i| 1 << i).collect();
v.extend([31, 57, 99, 163]);
```

### 4.12 partition

按闭包将条目分入两个集合：

```rust
let (living, nonliving): (Vec<&str>, Vec<&str>)
    = things.iter().partition(|name| name.as_bytes()[0] & 1 != 0);
```

要求结果类型实现 `Default + Extend`。

### 4.13 for_each / try_for_each

```rust
iter.for_each(|gift| println!("You have received: {}", gift));
iter.try_for_each(|gift| writeln!(&mut file, "{}", gift))?;
```

## 五、实现自己的迭代器

### 5.1 简单范围迭代器

```rust
struct I32Range { start: i32, end: i32 }

impl Iterator for I32Range {
    type Item = i32;
    fn next(&mut self) -> Option<i32> {
        if self.start >= self.end { return None; }
        let result = Some(self.start);
        self.start += 1;
        result
    }
}
```

实现了 `Iterator` 的类型自动获得 `IntoIterator` 实现，可直接用于 `for` 循环。

### 5.2 二叉树的中序遍历迭代器

```rust
struct TreeIter<'a, T> {
    unvisited: Vec<&'a TreeNode<T>>  // 栈：追踪未访问的节点
}

impl<'a, T: 'a> TreeIter<'a, T> {
    fn push_left_edge(&mut self, mut tree: &'a BinaryTree<T>) {
        while let NonEmpty(ref node) = *tree {
            self.unvisited.push(node);
            tree = &node.left;
        }
    }
}

impl<'a, T> Iterator for TreeIter<'a, T> {
    type Item = &'a T;
    fn next(&mut self) -> Option<&'a T> {
        let node = self.unvisited.pop()?;
        self.push_left_edge(&node.right);  // 下一次将遍历右子树的最左节点
        Some(&node.element)
    }
}
```

通过实现 `IntoIterator` 使 `&BinaryTree` 可迭代：

```rust
impl<'a, T: 'a> IntoIterator for &'a BinaryTree<T> {
    type Item = &'a T;
    type IntoIter = TreeIter<'a, T>;
    fn into_iter(self) -> Self::IntoIter { self.iter() }
}
```

## 六、关键要点

- **惰性求值**：适配器链条在调用 `next`（或 `for`/`collect` 等消费者）之前不执行任何工作
- **零成本抽象**：编译期内联，适配器链条与手写循环生成相同机器码
- **`size_hint`**：提供条目数估算，帮助 `collect` 等预分配正确大小
- 迭代器是 Rust 中最常用的惯用模式之一，能大幅提升代码表达力
