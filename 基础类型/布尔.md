# Rust 布尔类型

布尔类型是 Rust 中最简单的标量类型，只代表"真"或"假"两种状态。它用一个字节存储，是逻辑判断与控制流的基石。

## 一、速览

| 类型 | 可能的值 | 位宽 | 默认值 |
| :--- | :--- | :--- | :--- |
| `bool` | `true` 或 `false` | 1 字节 | `false` |

## 二、核心特性

`bool` 只有两个字面量：`true` 和 `false`，没有其他隐式转换的可能。

虽然理论上 1 个比特就够，但 Rust 将其对齐到 **1 个字节**，方便寻址。

在控制流方面，`if`、`while` 等语句的条件表达式必须返回 `bool` 类型，不接受 `0` 或非零值。

    if true {
        println!("条件为真");
    }

## 三、运算符与组合逻辑

布尔值主要通过逻辑运算符来组合，且 `&&` 和 `||` 具有**短路特性**——一旦能确定结果，就不再执行右边的表达式。

| 运算符 | 说明 | 示例 |
| :--- | :--- | :--- |
| `&&` | 逻辑与（短路） | `true && false` → `false` |
| `\|\|` | 逻辑或（短路） | `true \|\| false` → `true` |
| `!` | 逻辑非 | `!true` → `false` |

    let a = true;
    let b = false;

    println!("{}", a && b);   // false
    println!("{}", a || b);   // true
    println!("{}", !a);       // false

## 四、常见用途

**守卫条件**：判断输入是否合法、是否达到边界。

**标志位**：记录某种状态是否开启，例如 `let mut is_running = true;`。

**比较与判断**：所有比较运算符（`==`、`!=`、`<`、`>` 等）的返回值都是 `bool`。

    let x = 5;
    let is_positive = x > 0;  // is_positive 的类型是 bool，值为 true

## 五、与其他类型的转换

从 `bool` 到整数，可以通过 `as` 关键字转换。`true` 变为 `1u8`，`false` 变为 `0u8`。

    let t = true as u8;   // 1
    let f = false as u8;  // 0

从整数到 `bool`，**没有隐式或直接的 `as` 转换**，需要手动判断。

    let num = 0;
    let is_nonzero = num != 0;  // 正确方式
