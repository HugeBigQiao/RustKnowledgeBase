# Rust 字符类型

字符类型 `char` 是 Rust 中表达单个 Unicode 标量值的标量类型。它并不是简单的单字节，而是 4 字节宽，能够容纳从字母、数字到 emoji 在内的各种字符。

## 一、核心特性

`char` 用单引号包围，字面量可以是：

- 普通 ASCII 字符，如 `'A'`、`'z'`、`'9'`
- 转义字符，如 `'\n'`（换行）、`'\t'`（制表）、`'\\'`（反斜杠）
- Unicode 码点，如 `'Ω'`、`'🦀'`（螃蟹 emoji）

```rust
    let uppercase_a = 'A';
    let crab = '🦀';
    let newline = '\n';
```

每个 `char` 固定占用 4 字节，因为 Rust 的 `char` 对应的是 Unicode 标量值（从 `U+0000` 到 `U+D7FF` 以及 `U+E000` 到 `U+10FFFF`）。这与 C 语言的 1 字节 `char` 完全不同。

## 三、字符的表示与转换

从 `u8` 转换为 `char` 需要通过 `as` 并确保数值是有效 Unicode 码点：

```rust
    let byte: u8 = 65;
    let ch = byte as char;   // 'A'
```

反过来，`char` 可以转换为 `u32` 以获取其标量值：

```rust
    let ch = 'A';
    let code = ch as u32;    // 65
```

`char` 也可以转换为 `u8`，但会**截断**，只保留最低 8 位，可能丢失高位信息。

## 四、字符的方法

`char` 类型提供了一些有用的方法，用于判断字符类别或进行转换。

```rust
let ch = 'a';

// 判断字符类别
ch.is_alphabetic(); // true，是否为字母
ch.is_numeric();    // false，是否为十进制数字
ch.is_alphanumeric(); // true，是否为字母或数字
ch.is_whitespace(); // false，是否为空白字符
ch.is_lowercase();  // true，是否为小写
ch.is_uppercase();  // false，是否为大写

// 转换为大写：注意 to_uppercase() 返回迭代器，因为某些字符转换可能产生多个字符
let upper: String = ch.to_uppercase().collect(); // "A"

// 获取 UTF-8 编码所需的字节数
let utf8_len = ch.len_utf8();  // 1，'a' 用 1 字节表示

// 获取 UTF-16 编码所需的 16 位单元数
let utf16_len = ch.len_utf16(); // 1
```

## 五、字符与字符串的关系

`char` 是单个 Unicode 标量值，而 Rust 的 `String` 和 `&str` 则是 UTF-8 编码的字节序列。不能将 `char` 直接当作单字符字符串使用，但可以通过迭代器或字面量构建字符串。

```rust
let ch = 'A';

// 单个 char 可以通过以下方式转为 String
let s1: String = ch.to_string();      // 使用 to_string()
let s2 = ch.to_string();              // 类型推断为 String
let s3 = String::from(ch);            // 使用 String::from
let s4: String = ch.into();           // 使用 into()，通常需要类型标注

// 遍历字符串，将产生 char 序列
let greeting = "你好";
for c in greeting.chars() {
    println!("{}", c); // 依次打印 '你'、'好'
}
// 注意：字符串底层是 UTF-8 字节序列，直接索引可能得到字节而非完整字符
```

遍历字符串可以得到 `char` 序列：

```rust
let greeting = "你好";
for c in greeting.chars() {
    println!("{}", c); // 依次打印 '你'、'好'
}
```
