# Rust 元组类型

元组是将多个**类型可以不同**的值组合在一起的复合类型。与数组不同，元组的每个位置可以存放不同类型的值，且长度在编译时固定。元组分配在栈上，拥有其元素的所有权。

## 一、速览

| 特性 | 说明 |
| :--- | :--- |
| 存储位置 | 栈 |
| 长度 | 编译时固定，不可增长或缩小 |
| 元素类型 | 每个位置可以不同 |
| 是否拥有数据 | 是，拥有所有权 |
| 默认值 | 无（需显式初始化所有元素） |

## 二、元组操作

```rust
    // 元组的类型签名用圆括号包裹各元素的类型，如 `(i32, f64, char)`。

    // 显式声明类型
    let t1: (i32, f64, char) = (42, 3.14, 'A');

    // 类型自动推断
    let t2 = (1, "hello", true);

    // 创建空元组（单元值）
    let empty: () = ();

    // 元组通过**点号加索引**来访问元素，索引从 `0` 开始。

    let t = (42, "hello", 3.14);

    let first = t.0;   // 42
    let second = t.1;  // "hello"
    let third = t.2;   // 3.14
```

与数组不同，元组的索引必须是编译时常量，不能使用变量动态索引。

## 四、元组解构

```rust
    // 元组支持模式匹配解构，可以将所有元素一次性地绑定到独立变量上。
    let t = (42, "hello", 3.14);

    // 完全解构
    let (x, y, z) = t;
    // x = 42, y = "hello", z = 3.14

    // 使用 .. 忽略部分元素
    let (a, ..) = t;
    // a = 42

    let (.., c) = t;
    // c = 3.14

    let (a, _, c) = t;
    // a = 42, c = 3.14，中间元素被 _ 忽略
```

## 五、元组作为返回值

```rust
    // 函数可以返回元组，这是从函数中返回多个值的惯用方式。

    // 返回多个值
    fn stats(numbers: &[i32]) -> (i32, i32) {
        let sum = numbers.iter().sum();
        let count = numbers.len() as i32;
        (sum, count)
    }

    let nums = [1, 2, 3, 4];
    let (total, count) = stats(&nums);
    // total = 10, count = 4
```

标准库中的 `checked_add` 等方法也是通过返回元组来传递溢出信息的。

```rust
    // 溢出检查方法返回 (Result<T, OverflowError>)，其中 T 是溢出前的类型。
    let result = 100.checked_add(100);
    // result = Some(200)

    let result = 100.checked_add(1000);
    // result = None(OverflowError::Overflow)
```

## 六、单元类型 `()`

```rust
    // 不包含任何元素的空元组 `()` 被称为**单元类型**。它既是类型也是值，类似于其他语言中的 `void` 或 `nil`。
    // 函数如果没有显式返回值，默认返回 ()
    fn do_something() {
        println!("Hello");
    }

    let result: () = do_something(); // result 的类型是 ()
```

## 七、元组与数组的对比

|  | 元组 `(T1, T2, ...)` | 数组 `[T; N]` |
| :--- | :--- | :--- |
| 元素类型 | 可以各不相同 | 必须全部相同 |
| 长度 | 固定，由元素个数决定 | 固定，由 `N` 指定 |
| 访问方式 | `t.0`, `t.1` 等 | `arr[0]`, `arr[1]` 等 |
| 适用场景 | 少量不同类型值的临时组合 | 大量同类型值的序列 |
| 遍历 | 不能直接遍历 | 可以遍历 |
