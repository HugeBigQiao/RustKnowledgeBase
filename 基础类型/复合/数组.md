# Rust 数组类型

数组 `[T; N]` 是 Rust 中将多个相同类型的值分配在栈上的集合类型。与向量不同，**数组的长度是编译时已知且不可变的**，这使得它非常适合存放固定大小的数据。

## 一、速览

| 特性 | 说明 |
| :--- | :--- |
| 存储位置 | 栈 |
| 长度 | 编译时固定，无法在运行时增长或缩小 |
| 元素类型 | 必须全部相同 |
| 是否拥有数据 | 是，拥有所有权 |
| 默认值 | 取决于元素类型（整数默认 `0`，浮点默认 `0.0` 等） |

## 二、数组操作

数组的类型签名为 `[T; N]`，其中 `T` 是元素类型，`N` 是编译时常量。

```rust
    // 显式声明类型，方括号中写明元素类型和长度
    let arr1: [i32; 4] = [1, 2, 3, 4];

    // 类型自动推断
    let arr2 = [1, 2, 3];

    // 创建包含相同初始值的数组：分号前是初始值，分号后是长度
    let zeros = [0; 5];        // [0, 0, 0, 0, 0]
    let truth = [true; 3];     // [true, true, true]

    //访问与修改元素
    let mut arr = [1, 2, 3, 4, 5];

    // 索引访问，索引类型必须是 usize
    let first = arr[0];
    let second = arr[1];

    // 修改元素（数组必须是 mut）
    arr[0] = 10;

    // 越界索引会在编译时（常量索引）或运行时（变量索引）panic
    // let bad = arr[10];  // 运行时 panic: index out of bounds

    // 安全访问，返回 Option<&T>
    let maybe = arr.get(1);  // Some(&2)
    let none = arr.get(10);  // None

    //遍历数组
    let arr = [10, 20, 30, 40];

    // 遍历值（复制语义类型可以直接遍历，所有权语义类型需借用）
    for element in arr {
        println!("{}", element);
    }

    // 借用遍历（不消耗数组）
    for element in &arr {
        println!("{}", element);
    }

    // 带索引的遍历
    for (i, val) in arr.iter().enumerate() {
        println!("arr[{}] = {}", i, val);
    }
```

## 三、数组解构

```rust
    // 数组支持模式匹配和解构，可以将数组元素一次性地绑定到多个变量上。

    let arr = [1, 2, 3];

    // 解构数组
    let [a, b, c] = arr;
    // a = 1, b = 2, c = 3

    // 使用 .. 忽略剩余元素
    let [first, ..] = arr;
    // first = 1

    let [.., last] = arr;
    // last = 3
```

## 四、数组与切片的关系

```rust
    // 数组可以自动强制转换为切片 `&[T]`。这意味着任何接受切片引用的函数，都可以直接传入数组的引用。
    fn analyze(slice: &[i32]) {
        println!("length: {}", slice.len());
        println!("first: {:?}", slice.first());
    }

    let arr = [10, 20, 30];
    analyze(&arr);  // &[i32; 3] 自动转换为 &[i32]

    // 数组创建切片时不会复制数据，只是创建一个指向原数组数据的视图。

    let arr = [1, 2, 3, 4, 5];
    let view: &[i32] = &arr[1..4];  // [2, 3, 4]，不复制数据
```

## 五、常用方法

```rust
    // 数组的方法主要通过调用其实现的 `slice` 方法获得，因为数组可以自动解引用为切片。

    let arr = [3, 1, 4, 1, 5];

    arr.len();         // 元素个数（5）
    arr.is_empty();    // 是否为空（false）
    arr.first();       // 第一个元素，返回 Option<&T>
    arr.last();        // 最后一个元素，返回 Option<&T>
    arr.contains(&3);  // 是否包含某个值
    arr.iter();        // 返回迭代器

    // 注意：排序等方法需要可变切片 &mut [T]，数组需声明为 mut
    let mut mut_arr = [3, 1, 4, 1, 5];
    mut_arr.sort();  // 原地排序
```
